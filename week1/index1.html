Local Storage

Javascripts localStorage can be used to save key/value pairs in the web browser. The data is stored even if the browser is closed.
Syntax for saving is as follows: localStorage.setItem(“key”, “value”);
Values can then be accessed through grabbing the info via key
Syntax for this: var newVariable = localStorage.getItem(“key”);

Mobile first tips

Mobile first = better functionality
Loading must be quick (less steps = better)
Reachability matters when designing for touch
Increase speed by reducing image usage, use svg/webfonts, optimize CSS and JS files.

Have passwords toggle visibility (helps prevent errors when filling out on mobile)
Linking phone numbers on website make it much easier for mobile devices
