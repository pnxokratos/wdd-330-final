WDD330 Week 3 Notes

Object literals

An object in JavaScript is a self-contained set of related values and functions.

Method

If a property’s value is a function, it is known as a method.

Naming

If a property’s name doesn’t follow the rules for naming variables, it needs to be quoted.

Accessing properties

You can access the properties of an object using the dot notation. You can also access an object’s properties using bracket notation.

Calling methods

To call an object’s method we can also use dot or bracket notation. Calling a method is the same as invoking a function, so parentheses need to be placed after the method name

Checking if Properties or Methods Exist

The in operator can be used to check whether an object has a particular property. Another way is to use the hasOwnProperty()

Object.entries()

Returns an array consisting of enumerable property [key, value] pairs of the object which are passed as the parameter.

Object.values()

Returns an array whose elements are the enumerable property values found on the object

Object.keys()

The Object.keys() method will return an array of all the keys of any object that is provided as an argument

Adding Properties

New properties and methods can be added to objects at any time in a program. This is done by simply assigning a value to the new property.

Removing Properties

Any property can be removed from an object using the delete operator

Namespacing

Naming collisions occur when the same variable or function name is used for different purposes by code sharing the same scope. A solution to this problem is to use theobject literal patternto create a namespace for groups of related functions

DOM

Allows access to elements of a web page and enable interaction with the page by adding and removing elements, changing the order, content and attributes of elements, and even altering how they are styled.

DOM Shortcut methods

  • document.body
  • document.image
  • document.links
  • document.anchors
  • document.forms

Query selectors

  • document.querySelector()
  • document.querySelectorAll()

Event handlers

Event listeners

The recommended way of dealing with events, and the current standard, is to use event listeners

Event target

The target property returns a reference to the node that fired the event.

Coordinates of an event

  • screenX and screenY
  • clientX and clientY
  • pageX and pageY

Mouse events

  • mousedown
  • mouseup

Keyboard events

  • keydown
  • keypress
  • keyup

Stopping default behavior

preventDefault() is a method of the event object that can be used inside the callback function to stop the default behavior happening
